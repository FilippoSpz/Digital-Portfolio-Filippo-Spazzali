# 🌐 Digital Portfolio

A modern and responsive portfolio website to showcase projects, experiences, and professional skills.  
Official domain: **https://filippospz.netlify.app/**

---

## 📖 Description
This project was created with the goal of building a simple yet elegant online portfolio that allows me to:
- Present who you are and what I do
- Showcase my projects with descriptions and links
- Share my work and educational experiences
- Provide a quick way to get in touch

---

## 🛠️ Technologies Used
- **HTML5** for structure
- **CSS3** for styling
- **JavaScript** for interactivity
- **TypeScript** for stronger typing and robustness
